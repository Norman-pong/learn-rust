# Norman's Rust 笔记本

欢迎。这是 Norman 个人的 Rust 学习笔记仓库。

## 这是什么

- **不是公共教程**——内容面向已有编程基础的自己，假设你懂 JS/TS/Python/C++
- **TLDR 风格**——基础章用"一句话 + 对比表 + 踩坑清单"格式，不从头讲语法
- **错误驱动**——`compiler-pitfalls/` 收集真实编译错误案例（错码→报错→解释→修复）

## 怎么用

```bash
just serve    # 浏览器预览 http://localhost:3000
just test     # 运行全部练习（exercises/ 目录）
just clean    # 清理构建产物
just weekly   # 生成周报模板
```
